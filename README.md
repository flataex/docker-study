### Git branch & merge

- 각 스터디원 branch : 각자 정리한 내용 + 리팩토링한 kotlin project 관리 ex) `bghan`
- main branch &larr; 각 스터디원 branch PR 생성(월요일 00시까지)
- 코드 리뷰 위한 PR comment(수요일 14시까지)
- 리뷰할 내용 없으면 PR approve
